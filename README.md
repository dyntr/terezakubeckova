## Overview

Marketing and lead-generation site for **Ing. Tereza Kubečková**, an independent financial and wealth advisor in Prague (mortgages, investments, insurance, financial planning). Built as a single-page React app with a strong content focus: real licensing certificates, partner institution logos, and a set of genuinely useful financial calculators instead of generic stock copy.

## ✦ Features

- **10 interactive calculators** — mortgage, compound interest, savings, inflation, target savings, pension annuity, required rate of return, real return, income-protection reserve, and an amortization schedule, each with charts (Recharts) and code-split via `React.lazy`
- **Certificates section** — carousel of professional licensing certificates (consumer credit, life/non-life insurance, investment services…) with image + downloadable PDF proof for each
- **Reviews carousel** and a **photo gallery** (office, diplomas, event highlights)
- **Partner logos strip** — 24 banks/insurers/investment houses (Allianz, ČSOB, Komerční banka, Generali…)
- **Contact form** via Web3Forms, cookie-consent banner, and dedicated Cookies/GDPR pages (React Router)
- SEO-tuned `index.html` with Open Graph/Twitter meta

## 🛠 Built with

React 18 + TypeScript · Vite · Tailwind CSS · shadcn/ui (Radix primitives) · Framer Motion · Recharts · Embla Carousel · React Hook Form + Zod · React Router · Vitest

## 📦 Getting started

```bash
npm install
npm run dev      # local dev server
npm run build    # production build → dist/
```
